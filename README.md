# Cupcake House Demo

Live site: https://wwwearth2nxtt.github.io/cupcakehouse_demo/

A cupcake-themed landing page built with HTML, CSS, and JavaScript.

## Features
- Swiper.js carousel with coverflow effect
- Custom pink-themed UI
- Header with cart icon and hover effects
- Image-based product gallery
- Boxicons social icons

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Swiper.js
- Boxicons

## Project Structure
.
├── index.html
├── style.css
├── script.js
└── img/

## How to Run Locally
Open `index.html` in your browser.
